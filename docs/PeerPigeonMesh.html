Class: PeerPigeonMesh

PeerPigeonMesh()

new PeerPigeonMesh()

Main mesh networking class that provides WebRTC-based peer-to-peer communication with intelligent routing, distributed hash table, and optional cryptographic features.

Source:
  • PeerPigeonMesh.js, line 16
Example
// Create a new mesh instance
const mesh = new PeerPigeonMesh({
  enableWebDHT: true,
  enableCrypto: false
});

// Initialize and connect
await mesh.init();
await mesh.connect('wss://signaling-server.com/websocket');

// Listen for events
mesh.on('peerConnected', (peerId) => {
  console.log('Peer connected:', peerId);
});

// Send messages
mesh.sendMessage('peer-id', { type: 'hello', data: 'Hello World!' });
mesh.broadcastMessage({ type: 'announcement', data: 'Hello everyone!' });
